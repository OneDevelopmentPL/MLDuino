Because the Arduino Uno R3 has only 2KB of RAM, we cannot run standard ML frameworks. Instead, we train a model on a computer and export it as plain, hardcoded C logic. 

Here is how you do it using Python and `micromlgen`!

## Step 1: Install Dependencies
On your PC, you need `scikit-learn` and `micromlgen` (a tool that converts trained scikit-learn models to C code).
```bash
pip install scikit-learn micromlgen
```

## Step 2: Train & Export (Python Template)
Create a Python script on your computer to train a simple classifier (e.g., a Decision Tree) and generate the C code.

```python
from sklearn.datasets import load_iris
from sklearn.tree import DecisionTreeClassifier
from micromlgen import port

# 1. Load some example data (e.g., 3 features mimicking our 3 analog sensors)
# Here we just use the famous Iris dataset and take only 3 features for simplicity
X, y = load_iris(return_X_y=True)
X = X[:, :3] 

# 2. Train a Decision Tree Classifier
clf = DecisionTreeClassifier(max_depth=3)
clf.fit(X, y)

# 3. Export the model to C code
c_code = port(clf)

print("Copy the code below into your model.cpp:\n")
print(c_code)
```

## Step 3: Integrate with MLDuino

1. Run the Python script above.
2. It will output a C function (usually `predict()`).
3. Open `model.cpp` in this project.
4. Replace the dummy `run_model` logic with the logic generated by `micromlgen`. You may just need to wrap the `predict()` function inside `run_model()`.

Example `model.cpp` after integration:
```cpp
#include "model.h"

// --- GENERATED CODE FROM MICROMLGEN GOES HERE ---
int predict(float *x) {
    // ... generated decision tree if/else statements ...
    if (x[0] <= 0.5f) {
        return 1;
    }
    return 0;
}
// ------------------------------------------------

int run_model(float* features) {
    // Wrap the generated function here
    return predict(features);
}
```

## Step 4: Flash!
1. Open `MLDuino.ino` in the Arduino IDE.
2. Connect your Arduino Uno R3.
3. Click **Upload**.
4. Open the Serial Monitor (115200 baud) to see your AI in action!
